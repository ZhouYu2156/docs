# Geek Notes

## Introduction
> This is a Notes Projects built by VitePress.
> This is a Developping Project.


## 下载这个项目
```bash
git clone https://github.com/Geek-ch/geek-notes.git
```

## 项目运行
```bash
cd geek-notes
npm install
npm run dev
```

## 项目构建
```bash
npm run build
```

## 项目部署
```bash
npm run deploy
```
