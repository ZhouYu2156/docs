# HTML进阶知识

## 音/视频元素
```html
<audio src="" controls autoplay muted loop></audio>
<video src="" controls autoplay muted loop></video>
```

- 常用属性:

|  属性名  |       定义       |   类型   |
| :------: | :--------------: | :------: |
|   src    |  多媒体资源路径  | 普通属性 |
| controls | 是否显示播放控件 | 布尔属性 |
| autoplay |   是否自动播放   | 布尔属性 |
|   loop   |     循环播放     | 布尔属性 |
|  muted   |     静音播放     | 布尔属性 |
